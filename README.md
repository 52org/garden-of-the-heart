# ☘️ 마음을 전하는 텃밭 ☘️

당장 말을 전하지 못하고 시간이 지나면 까먹을 때가 있지 않나요?<br/>
전하고 싶지만 나중에 확인해줬으면 하는 말들이 있어요..

<br/>

글을 담아 전하고 싶은 사람의 텃밭에 심어주세요, 받은 사람이 가꾸고 확인해볼거에요!

<br/>


[텃밭 만들러 가기!](https://52org.github.io/garden-of-the-heart)
